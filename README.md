# Mobile-Architect-And-Programming
---
**Description**
● Languages and Frameworks: Java, XML, SQLite, Android Studio, OpenAI 
● Designed and implemented an Android application supporting user login, SQLite data storage, data visualization, AI 
assistant, and Algorithms for data analysis (trends and graphs). This app allows users to store weight entries with dates and 
images to reflect on their weight loss (or gain) journey. The dashboard features a line chart of their progress using Data 
Structure and Algorithm components like the Min/Max algorithm and using ArrayLists for the weight entries.
